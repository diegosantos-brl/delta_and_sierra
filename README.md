Delta & Sierra | Academic Project

Important: This repository is an Academic Project created for educational purposes only. It is not a commercial service or a real business entity.
📝 Project Overview

Delta & Sierra is a web-based project developed as part of a technical curriculum to explore the full lifecycle of a web application—from initial front-end construction to global domain configuration and hosting.

The main objective of this study was to implement a professional-grade interface while understanding the underlying infrastructure:

    DNS & Domain Management: Testing name resolution and propagation.

    Hosting Environments: Deploying static assets through modern PaaS (Vercel).

    Performance Optimization: Ensuring low latency and responsive design.

🛠️ Tech Stack & Learning Goals

    Front-end Foundations: HTML5, CSS3, and JavaScript.

    Deployment & CI/CD: Automated workflow via GitHub and Vercel.

    Infrastructure Concepts: Study of DNS (A, CNAME, MX records) and registrar comparison.
